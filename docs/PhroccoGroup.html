PhroccoGroup.php

¶

Phrocco Group processor class.

Whereas the base Phrocco class operates on a single file to produce a single output, This class looks after recursively iterating over a directory and creating an output for each file found.

@author Ross Riley

Iterate and Process

All the magic happens here, whils it looks complicated this is essentially a directory iterator that creates an array of filenames, with a corresponding Phrocco object that can look after rendering itself.

 
    public function process()
    {
        $dir_iterator = new PhroccoIterator($this->options["i"]);
        $iterator = new \RecursiveIteratorIterator($dir_iterator, \RecursiveIteratorIterator::SELF_FIRST);

        foreach ($iterator as $file) {
            if(!$iterator->isDot() && in_array($iterator->getExtension(), $this->extensions[$this->options["l"]])) {

                $base_path = $this->options["i"];
                $rpath = str_replace($base_path, "",$file->getPath());
                $phrocco = new Phrocco($this->options["l"], $file);

                if(!$this->options["o"]) $output_dir = $file->getPath();
                else $output_dir = $this->options["o"];

                if($rpath !=$file->getPath()) $output_dir.="/".$rpath;
