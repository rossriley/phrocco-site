Phrocco ~ Literate Documentation for PHP

Phrocco is a simple documentation generator for PHP files.

Based on the original, Docco for javascript this port produces similar HTML documents that display your comments intermingled with your code. All prose is passed through Markdown, and code is passed through Pygments to provide syntax highlighting.

How to install?

Easiest method is to install globally via composer

composer global require rossriley/phrocco:dev-master

Then you can either add your global composer binary folder to your path, if you haven't yet done this you can add this to your shell profile, assuming you're on a *nix style OS:

export PATH=$PATH:~/.composer/vendor/bin

If you don't want to do this, you can call the binary directly and run:

~/.composer/vendor/bin/phrocco
